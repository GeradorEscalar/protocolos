Política de Privacidade – Gerador Escalar Terapêutico

Última atualização: 30/10/2025

O aplicativo Gerador Escalar Terapêutico foi desenvolvido para funcionar em conjunto com um dispositivo externo controlado por um microcontrolador via conexão Bluetooth. A privacidade do usuário é uma prioridade, e esta política descreve como os dados são tratados.

1. Coleta de Dados

O aplicativo não coleta, armazena ou compartilha dados pessoais dos usuários. Todas as configurações e comandos enviados ao dispositivo são processados localmente e não são transmitidos pela internet, exceto nos casos descritos no item 3.

2. Permissões do Dispositivo

Para funcionar corretamente, o aplicativo solicita as seguintes permissões:

  • Bluetooth: utilizada para se comunicar com o microcontrolador que controla o dispositivo externo.
  • Localização: exigida pelo sistema Android para permitir o uso do Bluetooth. Essa permissão não é utilizada para rastrear ou registrar a localização do usuário.

Todas as permissões são utilizadas exclusivamente para o funcionamento técnico do aplicativo e não são usadas para fins de publicidade, análise ou coleta de dados pessoais.

3. Conectividade

Embora a maior parte da operação ocorra localmente entre o smartphone e o microcontrolador via Bluetooth, em alguns momentos o aplicativo pode se conectar à internet para baixar dados técnicos ou atualizações de configuração a partir de um servidor web seguro. Nenhuma informação pessoal do usuário é enviada durante esse processo.

4. Uso do Dispositivo Externo

O aplicativo é gratuito para download na Play Store, mas seu uso depende da aquisição de um dispositivo externo compatível com o microcontrolador. O app permite o envio de comandos personalizados que geram ondas quadradas específicas, aplicadas ao dispositivo para fins terapêuticos.

5. Compartilhamento de Dados

Este aplicativo não compartilha dados com terceiros. Nenhuma informação é enviada para redes sociais, serviços de análise ou plataformas externas, exceto os dados técnicos baixados conforme descrito no item 3.

6. Contato

Em caso de dúvidas sobre esta política de privacidade ou sobre o funcionamento do aplicativo, entre em contato com o responsável pelo app através do e-mail: contato@geradorescalar.com.br
